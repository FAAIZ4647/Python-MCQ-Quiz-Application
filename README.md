# Python MCQ Quiz Application

## Overview
This project is a simple console-based MCQ (Multiple Choice Questions) quiz application developed using Python.  
It presents multiple-choice questions to the user, checks the answers, and displays the final score at the end of the quiz.

---

## Project Description
The Python MCQ Quiz Application is designed to simulate a basic examination system.  
Questions are displayed one by one with four options, and the user selects the correct answer.  
The program evaluates the answers, updates the score, and finally shows the total score.

This project demonstrates the use of loops, conditional statements, and user input handling in Python.

---

## Features
- Multiple-choice questions with four options  
- Automatic answer checking  
- Score calculation  
- User-friendly console interaction  
- Final score display  

---

## Technologies Used
- Programming Language: Python  
- Tools: VS Code / IDLE  
- Concepts:
  - Loops
  - Conditional statements
  - User input and output
